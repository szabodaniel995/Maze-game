Welcome to the maze game!

On PC, use WASD keys to control the ball.
On mobile, use the built-in gyroscope to control the ball. Hint: lock the screen into landscape view.
Touch the green rectangle with the blue ball to complete the game.

The game has been built using vanilla JavaScript, CSS, and the MatterJs library.
